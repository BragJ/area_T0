ADPixirad
===========
An 
[EPICS](http://www.aps.anl.gov/epics)
[areaDetector](http://cars.uchicago.edu/software/epics/areaDetector.html)
driver for the Pixirad detectors from 
[PiXirad](http://pixirad.pi.infn.it/).

Additional information:
