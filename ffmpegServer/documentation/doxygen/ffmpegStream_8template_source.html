00009 # # \file 
00010 # # ffmpegServer contains a very simple http server based on
00011 # # <a href="http://www.nulllogic.ca/httpd">nullhttpd</a>. If a file named 
00012 # # \<port\>.jpg is requested for any ffmpegStream object named \<port\>, then it 
00013 # # will return the last NDArray received by the object compressed to jpeg. If 
00014 # # \<port\>.mjpg is requested then it will return an mjpg stream over http to the
00015 # # client. Otherwise, it will return an index page listing all the available 
00016 # # streams.
